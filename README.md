# Contributing

## Prerequisites

- Python 3.12
- [uv](https://docs.astral.sh/uv/) for dependency management
- [just](https://just.systems/) for running tasks
- [pre-commit](https://pre-commit.com/)

## Installation

```bash
just install
```

This will install:

 - Python dependencies, using `uv`
 - pre-commit hooks, using `pre-commit`
